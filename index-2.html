alert提示框

native://alert?data={title:"标题",msg:"提示信息"}
native://alert
  1. title：标题 选填
  2. msg：提示信息 必填

confirm确认框

native://confirm?data={title:"标题",msg:"提示信息"ok:"确定的文本"cancel:"取消的文本"}&callback=confirm
native://confirm
  1. title：标题 必填
  2. msg：提示信息 必填
  3. ok：确定的文本 默认“确定”
  4. cancel：取消的文本 默认“取消”
  5. callback：点击确认后的回调 必填

返回的data格式应为:

{
    code: 200,
    msg: "执行成功",
    data:"ok"//ok是点了确定,cancel是点了取消
}

prompt 输入提示框

native://prompt?data={title:"标题",msg:"提示信息",placeholder:"请输入文字"}&callback=prompt
native://prompt
  1. title：标题 必填
  2. msg：提示信息 选填
  3. value:默认文本 选填
  4. placeholder:文本框提示文本 选填
  5. callback：点击确认后的回调 必填

返回的data格式应为:

{
    code: 200,
    msg: "执行成功",
    data:"输入的内容"
}

toast 小提示弹窗

native://toast?data={time:3,msg:"提示信息"}
native://toast
  1. time：展示时间 选填，默认2s
  2. msg:提示消息 ,支持多行显示 \n 必填

actionsheet 操作选择

native://actionsheet?data={title:"请选择",list:[{text:选项一,value:"001"}]}&callback=actionsheet
native://actionsheet
  1. title:标题 选填
  2. list:选项数组 ,键值对 必填
  3. callback:回调,返回选中项

返回的data格式应为:

{
    code: 200,
    msg: "执行成功",
    data:{text:选项一,value:"001"}
}

modal 弹窗

native://modal?data={title:"标题",msg:"内容",img:"url",link:"url"}
native://modal
  1. title:标题
  2. msg：内容
  3. img:图片地址
  4. link:点击了解更多跳转地址

分享share

native://share?data={title:标题,desc:描述,link:分享地址,pic:图片}&callback=share
native://share
  1. title:标题 （必选）
  2. desc:描述 （必选）
  3. link:链接 （必选）
  4. pic:图片地址 不填不展示
  5. callback:分享点击确定或取消后的回调

返回的data格式应为:

{
    code: 200,
    msg: "执行成功",
    data:"ok"//ok|cancel 点了确定或取消
}

图片选择selectpic

native://selectpic?data={max:5}&callback=selectpic
native://selectpic
  1. max:当前可以选择的最大数量，重复选择不作累计 必填
  2. callback ：选择图片后的回调,返回图片的base64

返回的data格式应为:

{
    code: 200,
    msg: "执行成功",
    data:{localIds:[base64,base64]}
}

日期选择selectdate

native://selectdate?data={format:"yyyy-MM-dd"}&callback=selectdate
native://selectdate
  1. format:返回格式,yyyy-MM-dd 或 yyyy-MM-dd HH:mm
  2. callback:选择日期后的回调

返回的data格式应为:

{
    code: 200,
    msg: "执行成功",
    data:'2016-01-06'
}
